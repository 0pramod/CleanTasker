# CleanTasker • Contractors Module (Analytics Dashboards)

## Install & Run
```bash
npm install
npm run dev
```

- Contractor performance: `/contractors/contractor-performance`

- Contractor dashboard: `/contractors/contractor-dashboard`
- Manager dashboard: `/contractors/manager-dashboard`

Other pages:

- `/contractors/workforce`
- `/contractors/verification`
- `/contractors/compliance`
